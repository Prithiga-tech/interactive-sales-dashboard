# 📊 Interactive Sales Dashboard using R Shiny

## 📌 Project Overview

This project is an interactive data visualization dashboard built using R and Shiny Package.
It allows users to explore and analyze sales data dynamically through filters and visual charts.
The dashboard simulates a company's sales performance across different regions and product categories.

## 🚀 Features

* 🔎 Filter data by **Region** and **Product**
* 🎚️ Adjust **Price Range** using slider
* 📊 Dynamic visualizations:

  * Bar Chart (Sales by Region)
  * Scatter Plot (Price vs Quantity Sold)
  * Pie Chart (Sales Distribution by Product)
  * ⚡ Real-time updates based on user input

## 🛠️ Technologies Used

* R Programming
* Shiny (for interactive web apps)
* ggplot2 (for data visualization)
* dplyr (for data manipulation)

## ▶️ How to Run the Project

1. Install R and RStudio

2. Install required packages:
   install.packages("shiny")
   install.packages("ggplot2")
   install.packages("dplyr")

3. Open the project in RStudio

4. Run the file:
   app.R

5. Click **Run App**
